# Install
## Works on Void Linux only
[You should read the script before running it](https://blog.dijit.sh/don-t-pipe-curl-to-bash)
```sh
curl -LO "https://gitlab.com/cha0t1c/dotfiles/-/raw/master/install"
sh install
```
