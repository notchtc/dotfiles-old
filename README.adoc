== Install
Clone the repository
[source,sh]
----
git clone --recurse-submodules https://gitlab.com/cha0t1c/dotfiles.git $HOME/.dotfiles
----

Install https://github.com/thoughtbot/rcm[rcm]

Install the dotfiles
[source,sh]
----
RCRC=$HOME/.dotfiles/rcrc rcup
----
